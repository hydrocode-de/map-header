Viridis Colorscale

The map above demonstrates two things -
First, it uses customized colors created in Mapbox Studio, The Viridis colorscale is applied to the background map and the street features. Of course any other colors - exactly matching your brand colors - can be implemented into the Map.
Secondly, the streets are not imported as a GEOJson data source, but rather managed through Mapbox datasets. That makes the animations much more smooth as the data which to display is managed server side.

What is it?

Used libraries

This template uses Foundation CSS design. The map is powered by Mapbox GL, displaying street network build from OpenStreetMap data, distributed under ODbL 1.0 License.
